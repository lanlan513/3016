## 1. 产品概述

「视觉解构」是一个专注于视觉审美与设计分析的单页 Web 应用。用户上传任意图片（穿搭、建筑、画作等），AI 自动提取色彩调色板、线条构图与风格标签，帮助用户以结构化视角理解视觉作品的底层设计语言。

- 目标用户：设计师、摄影师、艺术爱好者、穿搭博主等对视觉审美有追求的创意人群
- 核心价值：将模糊的"审美直觉"转化为可量化、可复用的设计参数

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 访客 | 无需注册 | 上传图片并获取分析结果 |

### 2.2 功能模块

1. **主页面**：图片上传区、分析结果展示区（调色板 + 构图 + 风格标签）

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主页面 | Hero 区域 | 极简标题 + 副标题，呼吸感留白，品牌调性传达 |
| 主页面 | 图片上传区 | 拖拽或点击上传图片，支持预览，上传动画反馈 |
| 主页面 | 调色板提取 | 从图片中提取 5-8 个主色，展示色块 + HEX 值，点击可复制 |
| 主页面 | 构图分析 | 检测图片中的水平/垂直/对角线条分布，以叠加层可视化呈现 |
| 主页面 | 风格标签 | 基于色彩与构图特征生成 3-6 个风格标签（如"极简主义""暖调复古""高对比度"） |
| 主页面 | 分析动画 | 分析过程中展示优雅的加载动画，色彩粒子扩散效果 |

## 3. 核心流程

用户打开页面 → 拖拽/点击上传图片 → 图片预览确认 → 触发分析动画 → 依次展示调色板、构图分析、风格标签 → 用户可交互查看详情/复制色值

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：黑白灰中性色系（`#0A0A0A` / `#1A1A1A` / `#F5F5F5` / `#FFFFFF`）
- 强调色：无彩色强调，通过明度对比而非色相差异建立层级
- 按钮：细线描边按钮，hover 时填充过渡，圆角极小（2px）
- 字体：标题使用 Playfair Display（衬线体，传达编辑感与优雅），正文使用 DM Sans（无衬线，几何感，现代）
- 布局：单栏垂直流，大量留白，内容居中，最大宽度 960px
- 图标：Lucide 线性图标，1.5px 描边
- 微互动：色块 hover 放大 + 阴影、标签淡入交错动画、构图线条绘制动画、上传区域脉冲边框

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主页面 | Hero 区域 | 居中大标题 Playfair Display 48px，副标题 DM Sans 16px light，下方细线分隔 |
| 主页面 | 图片上传区 | 虚线边框矩形区域，拖拽高亮，中央图标+文字提示，上传后替换为图片预览 |
| 主页面 | 调色板提取 | 水平排列色块条，每个色块下方 HEX 值，hover 放大 + 浮层显示 RGB/HSL |
| 主页面 | 构图分析 | 图片叠加半透明线条层，检测到的主线条以白色描边动画绘制 |
| 主页面 | 风格标签 | 标签以胶囊形式排列，交错淡入动画，浅灰底+深色字 |

### 4.3 响应式

- 桌面优先设计，最大宽度 960px 居中
- 平板端：调色板色块缩小，构图分析保持原图比例
- 移动端：单列布局，上传区全宽，色块改为网格排列
